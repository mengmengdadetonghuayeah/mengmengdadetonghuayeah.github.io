发表于 2018-03-31 | 分类于 other |
  • 初始化项目
    • create-react-app demo
    • cd demo
    • yarn run eject 将配置文件暴露出来
    • yarn add －D babel-preset-es2015 babel-preset-stage-0 babel-preset-react babel-cli
      Babel(官网：https://babeljs.io/)是一个广泛使用的转码器，可以将es6语法转换为es5语法，从而在现有环境执行。
      es2015转码规则：
      npm install –save babel-preset-es2015
      react转码规则 ：
      npm install —save babel-preset-react
      es7不同阶段语法提案，选装一个 ：
      npm install –save babel-preset-stage-0
      npm install –save babel-preset-stage-1
      npm install –save babel-preset-stage-2
      npm install –save babel-preset-stage-3
      babel-cli工具，用于命令行转码。

  • 发布包

    • 发布到gitlab上

    • 发布到npm上
      如果你在npm有帐号可以采用npm adduser命令 输入用户名、密码、邮箱即可。
      如果没有，则去注册一个按同样方法执行即可。
      之后执行npm set registry http://registry.npmjs.org
      此条命令是将registry切回最初的源。因为最初的http://registry.npmjs.org这个源国内用户访问非常慢，所以一开始我们可能换成了国内淘宝的镜像：http://registry.npm.taobao.org。但是淘宝这个镜像没有开放上传的权限，因此会报这样的错误：

      最后sudo npm publish输入电脑密码即可上传到npm上，成功之后可去http://npmjs.com/package/查看你包的信息
      npm version <update_type>
      此条命令是后续更新包的操作，update_type填写你更新后的版本号，不能与之前的版本号重复，之后再执行npm publish即可。此方法上传的包为公共包，所有人均可看见。
      如果你想上传私有包的话，
      npm init –scope=你在npm上的用户名
      之后会出现一些包的信息，全部点击回车键即可，最后会出现一个类似于 @zhouhong/private-package这样形式的名字，之后npm publish即可（不过你需要有一个付费账号，每月是7美元，否则会提示你402错误）
      如果你想让其他成员也能使用该私有包，则使用以下命令
      npm owner 使用人的用户名 包名

  • 安装包
    • 发布到gitlab上的包的安装
    • 使用yarn add方法
      yarn add ssh://git@gitlab.dongwu-inc.com:10022/homework/npm-demo.git
      yarn add http://gitlab.dongwu-inc.com:10080/homework/npm-demo.git
    • 使用npm install 方法
      npm install–save git+ssh://git@gitlab.dongwu-inc.com:10022/homework/npm-demo.git
      npm install–save git+http://gitlab.dongwu-inc.com:10080/homework/npm-demo.git
      使用的方法：import { 组件 } from ‘包名’
    • 发布到npm上的包的安装
      如果是公共包
      npm install —save 包名
      使用方法同上
      如果是私有包
      npm install —save @zhouhong/private-package
      使用方法为
      import from {组件名} from ‘ @zhouhong/private-package’
