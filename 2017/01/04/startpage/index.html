IOS Android 应用名/图标/启动页设置

发表于 2017-01-04 | 分类于 react-native |

启动页（1）

添加图片到项目：右键 app，选择添加文件，把launcher图片添加到项目
选择LaunchScreen.xib文件，在右下角选择添加imageview，
如果尺寸不合适，选择图片，按住左键，分别拉向上、下、左、右四个方向，
选第一个按钮，然后把数值都设置成0（这样会铺满屏幕）如图：

启动页（2）

两种都配置开启应用会出现黑屏，该方式可以对应配置多种设配，可以控制启动页时间。
Image.xcassets / 加号 / App Icon & Launch Image / New iOS Launch Image

左侧会出现一个LaunchImage，这个就是我们刚才创建的文件。
右侧的勾选可以让你选择是否要对ipad、横屏、竖屏、以及低版本的ios系统做支持

点击一张图片的时候，右侧会出现Expected Size 1242 * 2208 pixels，
这个代表启动图片的尺寸，按照尺寸修改图片，然后将图片拖入即可。
